# Quran Kareem | القرآن الكريم

> A modern web application for reading and listening to the Holy Quran with a clean and comfortable interface.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-quran--kareem--ya--21.vercel.app-2563eb?style=for-the-badge&logo=vercel)](https://quran-kareem-ya-21.vercel.app/)

---

## 📖 Description

**Quran Kareem** is a modern web application designed to provide a clean, distraction-free experience for reading and listening to the Holy Quran online.

The website allows users to navigate easily between all **114 Surahs**, continue reading from their **last saved position**, and listen to Quran recitations directly from the browser — all within a fast, responsive interface optimized for both mobile and desktop devices.

The goal of the project is to provide a simple and elegant Quran reading experience that works smoothly on any device without unnecessary complexity.

---

## Features

- 📖 Read all **114 Surahs** of the Holy Quran
- 🔊 Listen to Quran recitation for each Surah directly from the website
- 📍 Automatically save the **last read position**
- 🔄 Smooth navigation between Surahs
- 📱 Fully responsive design for mobile and desktop
- 🌙 Dark reading interface for better eye comfort
- ⚡ Lightweight and fast performance

---

## ⚙️ How It Works

1. The homepage displays a list of Quran Surahs.
2. Selecting a Surah opens a dedicated reading page.
3. The page displays the full Surah text with clear Arabic typography.
4. Users can play the **audio recitation** of the Surah directly from the page.
5. The website automatically **saves the last read Surah** so users can continue later.
6. Navigation buttons allow users to move to the **next or previous Surah** easily.

## 💻 Running Locally

Clone the repository

git clone https://github.com/Youssef-Atef-20/Quran-Kareem.git

Navigate to the project folder

cd Quran-Kareem

Install dependencies

npm install

Run the development server

npm run dev

Build the project

npm run build

Preview the production build

npm run preview
